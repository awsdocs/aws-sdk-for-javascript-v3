**SDK for JavaScript V3**

```

// Set the AWS Region.

const snsClient = new SNSClient({ region: REGION });
export  { snsClient };
```

```

**SDK for JavaScript V3**  
Create the client in a separate module and export it\.

import  { SNSClient } from "@aws-sdk/client-sns";

const REGION = "REGION"; //e.g. "us-east-1"
